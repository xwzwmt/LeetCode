* {: id="20201210142415-m28cc38"}暴力法
{: id="20201210142415-72jpcfg"}

```
public static List<String> fizzBuzz(int n) {
        List<String> result=new LinkedList<>();
        for(int i=1;i<=n;i++){
            if((i%3==0)&&(i%5==0)){
                result.add("FizzBuzz");
                continue;
            }
            if(i%3==0){
                result.add("Fizz");
                continue;
            }
            if(i%5==0){
                result.add("Buzz");
                continue;
            }
            result.add(i+"");
        }
        return result;
    }
```
