* {: id="20201108101254-vc87cy4"}暴力法，双层for循环
{: id="20201108101254-c35k3pj"}

```
 int MaxA=0;
        for(int i=0;i<height.length-1;i++){
            for(int j=i+1;j<height.length;j++){
                int area=(j-i)*Math.min(height[i],height[j]);
                if(area>MaxA){
                    MaxA=area;
                }
            }
        }
        return MaxA;
```
