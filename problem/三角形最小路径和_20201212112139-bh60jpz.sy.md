#### [三角形最小路径和](https://leetcode-cn.com/problems/triangle/)
{: id="20201212112146-96ixorr"}

* 二维数组，从下到上

```
public static int minimumTotal(List<List<Integer>> triangle) {
        int m=triangle.size();
        int n=triangle.get(0).size();
        int [][] dp=new int[m][n];
        for(int i=m-2;i>=0;i++){
            for(int j=0;j<n;j++){
                dp[i][j]=Math.min(dp[i-1][j],dp[i-1][j-1]);
            }
        }
        return dp[0][0];
    }
```
