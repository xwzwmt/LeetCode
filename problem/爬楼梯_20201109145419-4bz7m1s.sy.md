* {: id="20201211105741-lwn4zm6"}优化递归。
{: id="20201211105741-izf4jwz"}

使用数组a[n]保存每次递归的结果，避免重复计算
{: id="20201211105741-ivzd1bi"}

```
 public int climbStairs(int n) {
        int a[]=new int[n+2];
        a[1]=1;
        a[2]=2;
        return climb(a,n);
    }
     public int climb(int a[],int n){
        if(a[n]>0){
            return a[n];
        }
        a[n]=climb(a,n-1)+climb(a,n-2);
        return a[n];
    }
```
