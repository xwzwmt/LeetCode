#### [二叉树的最小深度](https://leetcode-cn.com/problems/minimum-depth-of-binary-tree/)

* 递归

```
 public int minDepth(TreeNode root) {
        if(root==null){
            return 0;
        }        int l= 0,r=0;
        l=minDepth(root.left);
        r=minDepth(root.right);

        if (l != 0 && r != 0) {
            return Math.min(l, r) + 1;
        } else if (r == 0) {
            return l + 1;
        } else {
            return r + 1;
        }
    }
```
